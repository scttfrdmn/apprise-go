🎯 Quick Start

Welcome to the Apprise-Go dashboard! This interface allows you to manage your notification services, send notifications, and monitor your system.


# Send a notification via API curl -X POST http://localhost:8080/api/v1/notify \ -H "Content-Type: application/json" \ -d '{"body": "Hello World!", "urls": ["webhook://httpbin.org/post"]}' # List all supported services curl http://localhost:8080/api/v1/services # Check server health curl http://localhost:8080/health

Key Endpoints:

GET /health - Health check GET /api/v1/services - List supported services POST /api/v1/notify - Send single notification POST /api/v1/notify/bulk - Send multiple notifications GET /api/v1/scheduler/jobs - List scheduled jobs POST /api/v1/scheduler/jobs - Create scheduled job GET /api/v1/config - Get server configuration
